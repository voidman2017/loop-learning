# 2026-07-23 学习日报

## 今日学习综述

今日启动了 Vue Router 课题的系统学习。基于官方文档与社区资源，梳理了前端路由的核心概念与 Vue Router 的关键特性。

## 关键发现与结论

1. **Vue Router 核心机制**：基于 hash 模式或 history 模式实现 URL 与组件的映射。hash 模式利用 `hashchange` 事件，history 模式利用 HTML5 History API（`pushState`/`replaceState`）。（来源：Vue Router 官方文档 - [Getting Started](https://router.vuejs.org/guide/)）
2. **动态路由匹配**：使用 `:param` 语法定义动态片段，匹配到的参数通过 `$route.params` 获取。
3. **导航守卫**：分为全局守卫（`beforeEach`、`beforeResolve`、`afterEach`）、路由独享守卫（`beforeEnter`）和组件内守卫（`beforeRouteEnter`、`beforeRouteUpdate`、`beforeRouteLeave`）。执行顺序：全局 beforeEach → 路由独享 beforeEnter → 组件内 beforeRouteEnter → 全局 beforeResolve → 全局 afterEach。
4. **路由懒加载**：结合 Vue 的异步组件与 Webpack 动态导入 `import()` 实现组件按需加载。

## 知识盲区识别

- 导航守卫中 `next` 函数的详细行为与错误处理
- 路由元信息 `meta` 的实战应用模式
- 动态路由匹配的优先级规则（`/user/:id` 与 `/user/profile` 共存时的匹配顺序）
- 嵌套路由与命名视图的结合使用

## 待办事项

- [ ] 阅读 Vue Router 源码仓库（`vuejs/router`），重点关注 `createRouter` 与 `createWebHistory` 实现
- [ ] 实现一个简化版前端路由，覆盖 hash 模式与 history 模式
- [ ] 输出 Vue Router 核心原理研究笔记
- [ ] 在项目中实践动态路由与导航守卫的复杂场景

---

> 来源：Vue Router 4.x 官方文档、Vue.js 官方指南
